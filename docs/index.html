说明

本页面展示的是已经验证过的缓存演示，用于稳定交付网页链接。 本地版本还提供 FastAPI 页面，可以触发真实的 Habitat + OWL-ViT + Agent 运行。

当前最终展示目标为沙发和椅子。系统接口按多目标方式设计，后续可以通过切换到包含相应物体且适合展示的场景，继续扩展到床、桌子、厨房台面等更多目标。本次提交选用已经完成验证的沙发和椅子作为主展示目标。

当前任务是导航任务，机器人双臂固定不参与控制。

## 机器人形态 当前机器人建模为**轮式双臂移动机器人导航抽象**：轮式底盘负责导航，双臂固定不参与本导航任务；第一视角 RGB-D 相机安装在机器人上。当前展示聚焦导航能力，不涉及机械臂操作控制。
